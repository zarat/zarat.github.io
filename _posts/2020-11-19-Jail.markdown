Jail has an object-oriented, c-like syntax similar to Javascript. Visit the [documentation](https://jail.lima-city.at) for detailed information.

## Datatypes
Strings are enclosed in double quotes `"I am a string"`, Chars in single quotes `'a'`, arrays in brackets `[1, 3.14, 'a', "Hello world"]` and objects in curly braces `{ str: "Hello world", num: 3.14 }`. Numbers are not enclosed at all.

## Variables 
Variables are declared using the keyword `var`. Uninitialized variables has a value of `undefined`.

``` Javascript
var i; // declared, but undefined
i = 1; // defined

var ch = 'a'; // Char
var int = 1; // Integer
var hex = 0xab; // Hexadecimal integer
var bin = 10010b; // Binary integer
var oct = 101o; // Octal Integer
var dbl = 3.14159; // Double
var str = "Hello world"; // String
var arr = [1,2,3]; // Array

var a, b, c = 1; // a and b are undefined, c is initialized to 1
var a = b = c = 1; // a, b and c are initialized to 1
var a = 1, b = 2, c = 3; // a is initialized to 1, b to 2 and c to 3
```

## Functions
Functions are defined in 2 ways using the keyword `function`.

```Javascript
// funtion declaration
function add(a, b) { return a + b; } 

// function expression 
var sub = function(a, b) { return a - b; };
```

Functions are recursive which means a function can call itself during its execution. The process may repeat several times, outputting the result and the end of each iteration.

```Javascript
function count(num) {

    if(num <= 0) 
        return "num must be a positive integer";

    if(num > 10) 
        return "done!";

    print("count.. "); 
    return count(num + 1);

}

print(count(1));

// output
// count.. count.. count.. count.. count.. count.. count.. count.. count.. count.. done!
```

## Classes
Classes are defined using the keyword `class`. When there is a function defined inside the class with the same name as the class, this function is called the `constructor` and is executed automatically whenever a new instance has been created.

```Javascript 
class Test {
    Test: function(param) {
        this.name = param.name;
        this.data = param.data; 
    },
    name: "Test class",
    data: 0
};

var test = new Test({ name: "My new Class", data: 5 });
print(test.data);
```
